*****FUNCIONES PREDEFINIDAS DE PHP*****

 array_push():
 La función array_push se utiliza para agregar uno o más elementos al final de un array. Su propósito principal es aumentar la longitud del array y agregar nuevos elementos al final de la lista.

 intval():
 La función intval se utiliza para obtener el valor entero de una variable. Su propósito principal es convertir el valor de una variable a un número entero.

 count():
 La función count se utiliza para contar el número de elementos en un array o el número de propiedades en un objeto.

 rand():
 La función rand se utiliza para generar un número entero aleatorio. 
 
 str_repeat():
 La función str_repeat se utiliza para repetir una cadena (string) un número específico de veces.
 
 strtolower():
 La función strtolower se utiliza para convertir todos los caracteres de una cadena (string) a minúsculas.

 number_format():
 La función number_format se utiliza para formatear números con decimales, añadiendo separadores de mil y especificando el número de decimales que deseas mostrar. 
 
 ctype_alpha():
 La función ctype_alpha se utiliza para comprobar si todos los caracteres de una cadena (string) son letras del alfabeto.

 strcmp():
 La función strcmp se utiliza para comparar dos cadenas (strings).

 uasort():
 La función uasort se utiliza para ordenar un array asociativo utilizando una función de comparación definida por el usuario. 
 
 print_r():
 La función print_r en PHP se utiliza para imprimir información fácilmente legible sobre una variable en un formato que es principalmente utilizado para la depuración y desarrollo. Puede imprimir información sobre variables escalares, arrays y objetos.

 is_numeric():
 La función is_numeric se utiliza para determinar si una variable es un número o una cadena numérica. Su propósito es verificar si el valor de una variable es numérico o puede ser convertido a un número.

 strlen():
 La función strlen se utiliza para obtener la longitud (número de caracteres) de una cadena (string).

 strtoupper():
 La función strtoupper se utiliza para convertir todos los caracteres de una cadena (string) a mayúsculas.

 strpos():
 La función strpos se utiliza para encontrar la posición de la primera ocurrencia de una subcadena (string) dentro de otra cadena. 

 str_split():
 La función str_split se utiliza para dividir una cadena (string) en un array de caracteres.

in_array():
La función in_array() verifica si un valor específico está presente en un array. Devuelve true si el valor se encuentra en el array y false si no se encuentra.

empty():
La función empty() se utiliza para comprobar si una variable está vacía o no está definida. Devuelve true si la variable está vacía, false en otros casos, como por ejemplo: la variable no existe, la variable tiene un valor igual a 0, o bien, un string vacío, es null, es un array vacío.
